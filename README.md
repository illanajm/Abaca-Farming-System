Abaca Farming System

an AI solution about the current dilemma facing by PhilFida government in monitoring the abaca farmers data analytics and reports.

Need to have in your PC/Laptops
1. vscode
2. xampp
3. mySQL
4. python

Run the following in your Command Line or Terminal in vscode
1. py -m pip install streamlit pandas sqlalchemy pymysql plotly
2. py -m pip install --upgrade streamlit
3. py -m pip install -r requirements.txt
4. py -m pip install streamlit-extras
5. py -m pip install reportlab
6. py -m pip install openpyxl
7. py -m pip install matplotlib
8. py -m pip install kaleido
9. python database.py

To run the system after installation:
* run xampp and start apache and mysql
* run 'py -m streamlit run app.py' at the terminal
* if you have to reset the database for fresh start just run 'python reset_db.py' to delete all data in the database and apply changes in table if there is a change.


when resetting database run "python reset_db.py" but be carefull for running this command means deleting also the data stored in db.
