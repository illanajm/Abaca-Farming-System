Abaca Farming System

Need to have in your PC/Laptops
1. vscode
2. xampp
3. mySQL

Run the following in your Command Line or Terminal in vscode
1. pip install stremlit pandas sqlalchemy pymysql plotly
2. pip install --upgrade steamlit
3. pip install -r requirements.txt
4. pip install streamlit-extras
5. pip install reportlab
6. pip install openpyxl
7. pip install matplotlib
8. pip install kaleido
9. python database.py

when resetting database run "python reset_db.py" but be carefull for running this command means deleting also the data stored in db.

to run the system type "streamlit run app.py"
